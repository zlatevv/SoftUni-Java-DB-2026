# SoftUni Java DB 2026 Course

This repository contains course materials and exercise solutions for the **SoftUni Java DB 2026** track. The content is organized by modules, with each folder representing a topic area and its related tasks.

## Course Modules

### MySQL
Hands-on SQL practice covering the full lifecycle of working with a relational database:

- **Data Definition and Datatypes** – schema creation and core SQL types.
- **Basic CRUD** – create, read, update, and delete operations.
- **Built-in Functions** – string, numeric, date, and other function usage.
- **Data Aggregation** – grouping, aggregation, and analytics with SQL.
- **Table Relations** – keys, constraints, and relational modeling.
- **Subqueries and Joins** – multi-table querying patterns.
- **Database Programmability and Transactions** – procedures, functions, triggers, and transactions.
- **Exam Prep / Exam** – practice sets and final assessments.

### Spring Data
Module placeholder for Spring Data exercises and project work.

- Use the materials as references while solving tasks or preparing for exams.
